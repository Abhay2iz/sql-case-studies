## Tools Used

- **Database:** Microsoft SQL Server (SSMS)
- **Language:** T-SQL

## About

Each case study folder contains:
- A `README.md` with the problem statement and a summary of what was solved
- A `schema/` folder with table creation scripts
- A `solutions.sql` file with all queries, fully commented and ready to run

## How to Use

1. Open `schema/create_tables.sql` in SSMS and execute it to set up the database and tables.
2. Open `solutions.sql` and run the queries individually or as a batch.
